# Portfolio

A single-page animated portfolio landing built with React, TypeScript, Vite, and Framer Motion.

## Tech Stack

- React 19 + TypeScript
- Vite 8
- Tailwind CSS 4
- Framer Motion

## Project Structure

```text
src/
  App.tsx                 # Main animation scene and layout
  main.tsx                # App entry point
  index.css               # Global styles and base font imports
  assets/
    *.png                 # Image cards used in animation
    frames/               # Reference frames for animation timing/layout
```

## Getting Started

```bash
pnpm install
pnpm dev
```

## Scripts

- `pnpm dev` - start Vite dev server
- `pnpm build` - type-check and build production output
- `pnpm preview` - preview production build
- `pnpm lint` - run ESLint

## Output

Production files are generated in `build/`.
